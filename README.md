# 📩 Spam Ham Message Detection

A Machine Learning based web application that classifies SMS messages as **Spam** or **Ham (Not Spam)** using Python, Flask, Scikit-learn, and TF-IDF Vectorizer.

---

## 🚀 Features

- Detects Spam and Ham SMS messages
- Machine Learning based prediction
- Simple and user-friendly web interface
- Built with Flask
- Fast and accurate predictions

---

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- Pickle

## ⚙️ Installation

```bash
git clone https://github.com/manishasaini28/Spam-Ham-Detection.git

cd Spam-Ham-Detection

pip install -r requirements.txt

python app.py
```

---

## 📸 Output

- Enter an SMS message.
- Click **Predict**.
- Result will be displayed as:
  - ✅ HAM MESSAGE
  - 🚫 SPAM MESSAGE
